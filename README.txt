TaskNest AI – Team Task Manager

Overview:
TaskNest AI is a full-stack task and project management platform designed for AI operations teams. The application supports project management, task assignment, team collaboration, and role-based access control.

Features:
- User Authentication
- Role-Based Access Control (Admin/Member)
- Project Management
- Task Assignment & Tracking
- Dashboard Analytics
- Overdue Task Monitoring
- Responsive UI
- REST APIs
- PostgreSQL Database

Tech Stack:
Frontend:
- React
- TypeScript
- TailwindCSS
- Vite

Backend:
- Node.js
- Express.js
- PostgreSQL
- Prisma ORM

Deployment:
- Railway
